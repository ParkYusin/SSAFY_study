## 재귀함수
- 함수 내부에서 자기 자신을 호출하는 함수
- ex - 팩토리얼 
```python 
    def sol (n):
    print(f"{'-' * n} {n}")
    if n == 1:
        return n
    else :
        return n  * sol(n - 1)
print(sol(5))

# sol(5) -> 5 * sol(4):
# sol(4) -> 4 * sol(3)
# sol(3) -> 3 * sol(2)
# sol(2) -> 2 * sol(1)
# sol(1) -> 1

```

## packing, unpacking
- 패킹 예시 여러개의 값을 하나의 변수에 묶어서 담는 것
- numbers = [1,2,3,4,5]
- a, *b, c = numbers a 는 1 b는 2,3,4 c는 5
- '*'는 '에스터리스크'라고 말함
- 언패킹때 *하나는 리스트, 튜블 등 이터러블 ** 두개는 dict 에서만
